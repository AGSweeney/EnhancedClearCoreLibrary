# New Files Added for Coordinated Motion and Unit Support

This document lists all files that were created for the coordinated motion features (arcs and linear moves) and unit conversion support that do not exist in the original ClearCore library codebase.

**Note**: Documentation files (*.md) in the `docs/` folder are excluded from this listing.

## Header Files

### libClearCore/inc/

1. **ArcInterpolator.h**
   - Core arc interpolation engine class
   - Defines `ArcSegment` structure and `ArcInterpolator` class
   - Provides arc initialization and step generation methods
   - Fixed-point Q15 mathematics interface
   - **Path**: `libClearCore/inc/ArcInterpolator.h`

2. **CoordinatedMotionController.h**
   - High-level controller for coordinated motion
   - Manages arc queue and motor coordination
   - Public API for arc commands
   - Integration with MotorDriver system
   - **Path**: `libClearCore/inc/CoordinatedMotionController.h`

3. **TrigLUT.h**
   - Trigonometric lookup table interface
   - Fast sin/cos calculation functions for fixed-point angles
   - Q15 format support
   - **Path**: `libClearCore/inc/TrigLUT.h`

4. **UnitConverter.h**
   - Unit conversion system interface
   - Defines `UnitType` and `FeedRateUnit` enums
   - Defines `MotorMechanicalConfig` structure
   - Provides unit conversion class for physical units (inches, mm, degrees)
   - **Path**: `libClearCore/inc/UnitConverter.h`

5. **LinearInterpolator.h**
   - Linear interpolation engine for coordinated linear moves
   - Generates coordinated step commands for straight-line paths
   - Constant velocity along path
   - Fixed-point Q15 mathematics interface
   - **Path**: `libClearCore/inc/LinearInterpolator.h`

## Implementation Files

### libClearCore/src/

4. **ArcInterpolator.cpp**
   - Arc interpolation algorithm implementation
   - Fixed-point Q15 arithmetic for precision
   - Parametric arc path generation
   - Constant tangential velocity control
   - **Path**: `libClearCore/src/ArcInterpolator.cpp`

5. **CoordinatedMotionController.cpp**
   - Coordinated motion controller implementation
   - Arc queue management (up to 8 arcs)
   - Motor coordination logic
   - ISR integration (`UpdateFast()`)
   - **Path**: `libClearCore/src/CoordinatedMotionController.cpp`

6. **TrigLUT.cpp**
   - Trigonometric lookup table implementation
   - Linear interpolation between table entries
   - Sin/cos calculation functions
   - **Path**: `libClearCore/src/TrigLUT.cpp`

7. **TrigLUT_Data.h**
   - Generated sine lookup table data
   - 1024 entries covering 0 to 2π
   - Pre-computed values: `sin(i × 2π / 1024) × 32768`
   - Generated by Python script
   - **Path**: `libClearCore/src/TrigLUT_Data.h`

8. **UnitConverter.cpp**
   - Unit conversion implementation
   - Converts between physical units and step counts
   - Handles distance units (inches, mm, cm, meters, revolutions, degrees)
   - Handles feed rate units (inches/min, mm/min, mm/sec, RPM, etc.)
   - Calculates conversion factors from mechanical parameters
   - **Path**: `libClearCore/src/UnitConverter.cpp`

9. **LinearInterpolator.cpp**
   - Linear interpolation algorithm implementation
   - Fixed-point Q15 arithmetic for precision
   - Straight-line path generation with constant velocity
   - Coordinate step generation for two motors
   - **Path**: `libClearCore/src/LinearInterpolator.cpp`

## Example Application

### Microchip_Examples/ClearPathModeExamples/ClearPath-SD_Series/

10. **CoordinatedArcMoves/CoordinatedArcMoves.cpp**
   - Complete example application for arc moves
   - Demonstrates single arc moves
   - Shows continuous arc chaining
   - Includes error handling and status reporting
   - **Path**: `Microchip_Examples/ClearPathModeExamples/ClearPath-SD_Series/CoordinatedArcMoves/CoordinatedArcMoves.cpp`

11. **CoordinatedMovesWithUnits/CoordinatedMovesWithUnits.cpp**
   - Complete example application demonstrating unit support
   - Shows single motor moves in physical units (inches, mm)
   - Demonstrates coordinated linear moves with units
   - Demonstrates coordinated arc moves with units
   - Shows feed rate programming in physical units
   - Includes continuous move chaining
   - **Path**: `Microchip_Examples/ClearPathModeExamples/ClearPath-SD_Series/CoordinatedMovesWithUnits/CoordinatedMovesWithUnits.cpp`
